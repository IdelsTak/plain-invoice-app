# Security Policy

## Supported versions
This project is in early development. Security fixes are applied to the default branch.

## Reporting a vulnerability
Please do not publish vulnerabilities in public issues first.

Report with:
- Summary of the issue
- Impact and exploitability
- Reproduction steps
- Suggested mitigation if available

Use GitHub Security Advisories if available for this repository, otherwise contact maintainers directly.

## Response expectations
- Initial acknowledgment: within 3 business days
- Triage and severity assessment: as soon as reproducible details are available
- Fix timeline: based on severity and release readiness
